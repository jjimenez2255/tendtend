# Clawdbot DFW - Quick Start Checklist
## Get Live in 7 Days

### Day 1: Accounts & Tracking Setup
- [ ] **Google Ads Account**
  - Go to ads.google.com
  - Create new account with business email
  - Add payment method (credit card)
  - Set time zone to Central
  
- [ ] **Meta Business Suite**
  - Go to business.facebook.com
  - Create Business Account
  - Create or claim "Clawdbot DFW" Facebook Page
  - Create Instagram Business account
  - Add payment method

- [ ] **Google Analytics 4**
  - Go to analytics.google.com
  - Create property for clawdbotdfw.com
  - Get Measurement ID (G-XXXXXXX)
  - Add to website (or via Tag Manager)

- [ ] **Google Tag Manager**
  - Go to tagmanager.google.com
  - Create container for clawdbotdfw.com
  - Install container code on website

### Day 2: Pixel & Conversion Setup
- [ ] **Meta Pixel**
  - In Meta Business Suite → Events Manager
  - Create new Pixel
  - Install on website (via Tag Manager recommended)
  - Set up standard events: PageView, Lead
  
- [ ] **Google Ads Conversion Tracking**
  - In Google Ads → Goals → Conversions
  - Create conversion action: "Form Submission"
  - Create conversion action: "Phone Call"
  - Install via GTM

- [ ] **Call Tracking**
  - Sign up for CallRail ($45/month plan)
  - Create tracking number for website
  - Enable dynamic number insertion
  - Connect to Google Ads & Google Analytics

### Day 3: Landing Page Verification
- [ ] **Website Ready Check**
  - [ ] Clear headline explaining service
  - [ ] Obvious CTA button ("Get a Free Demo" / "Schedule Consultation")
  - [ ] Working contact form
  - [ ] Phone number visible
  - [ ] Mobile-responsive
  - [ ] Fast loading (<3 seconds)
  - [ ] SSL certificate (https://)
  - [ ] Basic trust elements (professional design)

### Day 4: Google Ads Campaign Launch
- [ ] **Create Campaign: Search - High Intent**
  - Campaign type: Search
  - Goal: Leads
  - Location: Dallas-Fort Worth-Arlington, TX (+25 miles)
  - Budget: $50/day
  - Bid strategy: Manual CPC (start at $10)
  
- [ ] **Ad Group 1: AI Automation**
  - Keywords:
    - "ai automation for small business dallas" [exact]
    - "ai business automation fort worth" [exact]
    - "business automation service dfw" [phrase]
    - ai automation small business dallas [broad]
  
- [ ] **Ad Group 2: Virtual Assistant Alternative**
  - Keywords:
    - "virtual assistant alternative dallas" [exact]
    - "ai assistant for business dallas" [phrase]
    - "automated assistant small business" [phrase]

- [ ] **Write Ads (3 variations)**
  - Use headlines from GTM plan
  - Include location
  - Include pricing if space allows
  - Strong CTA

- [ ] **Negative Keywords**
  - free, cheap, diy, jobs, hiring, salary, course, tutorial, chatgpt, openai

### Day 5: Meta Ads Campaign Launch
- [ ] **Create Campaign: Lead Generation**
  - Objective: Leads
  - Budget: $40/day
  - Placements: Automatic (or Facebook Feed + Instagram Feed)
  
- [ ] **Audience Setup**
  - Location: Dallas-Fort Worth area
  - Age: 28-60
  - Interests: Small business, Entrepreneurship, QuickBooks, Scheduling software
  - Custom: Website visitors (if pixel has data)

- [ ] **Ad Creative**
  - [ ] 1 image ad with bold statement
  - [ ] 1 carousel showing use cases
  - [ ] 1 simple video (even phone-recorded demo)

- [ ] **Lead Form**
  - Pre-filled fields: Name, Email, Phone
  - Custom question: "What type of business do you run?"
  - Thank you screen with calendar link

### Day 6: Monitoring & Optimization
- [ ] **Create Tracking Spreadsheet**
  - Date | Platform | Impressions | Clicks | Leads | Cost | CPL
  
- [ ] **Set Up Alerts**
  - Google Ads: Daily budget alert
  - Meta: Ad rejection notifications
  - CallRail: New call notifications to phone

- [ ] **Lead Response Process**
  - Goal: Respond to all leads within 1 hour during business hours
  - Script/template for first response
  - Demo scheduling link ready

### Day 7: Review & Expand
- [ ] **Check All Tracking Working**
  - Submit test form → appears in CRM/spreadsheet
  - Test call → appears in CallRail
  - Conversions showing in Google Ads
  - Conversions showing in Meta

- [ ] **Review Initial Performance**
  - Are ads showing?
  - CTR above 2%?
  - Any ad rejections to fix?

- [ ] **Plan Week 2 Tests**
  - Additional ad copy variations
  - New audience tests
  - Budget adjustments
